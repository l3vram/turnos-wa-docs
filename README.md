# turnos-wa · documentación pública

Sitio estático con la documentación de **turnos-wa**, un asistente de reservas y
turnos por WhatsApp para peluquerías, barberías y estética.

Publicado con GitHub Pages. Páginas:

- **`index.html`** — portada / hub.
- **`capacidades.html`** — qué hace y qué no (para evaluar si encaja con un negocio).
- **`negocio.html`** — producto, propuesta de valor, modelo de cobro, estado y roadmap.
- **`tecnica.html`** — arquitectura y decisiones de diseño (para colegas).

> Este repo contiene **solo la documentación pública**. El código de la aplicación
> vive en un repositorio privado aparte.
